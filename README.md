# Orgath

## What is it?
Orgath's purpose is to be a personnal organiser. It's inspired on the android's widget system.

## Technologies
- React
- Sass
- Webpack

## Api used
[Weather API](https://openweathermap.org/)
[Quote API](https://quotes.rest/)
